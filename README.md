Project MONTE (SJDM)
A GIS-based Public Transit Routing System for San Jose del Monte, Bulacan.

Project MONTE is a Research Capstone Project designed to help commuters visualize public vehicle routes based on their specific origin and destination. It utilizes spatial data to map local transit pathways (Jeepneys/Tricycles) for academic and civic research purposes.

⚠️ Important Disclaimers
Non-Official Status: This is a student-led research project. It is not an official mapping or transportation platform sanctioned by the Local Government Unit (LGU) of San Jose del Monte or the LTFRB.

Non-Commercial: This project is strictly for academic purposes and is not for profit or money promotion.

Usage Policy: No financial operations or commercial integrations are permitted within the MONTE framework without explicit authorization from the original developers.

🛠️ Technical Overview
Mapping Engine: Built with Leaflet.js

Data Processing: QGIS for routing and spatial analysis

Hosting: GitHub Pages
